== Description ==

Onswipe Link connects your WordPress installation to Onswipe.

== Installation ==

1. You can either put the contents of this folder in /site/wp-content/plugins/onswipe-link/
   OR
   Go to your WordPress Admin > Plugins > Add New > Upload and upload the .zip file.
2. Activate the plugin through WordPress Admin > Plugins > Onswipe Link > Activate.
3. Go to the Onswipe Link page on the WordPress Admin sidebar.
4. If you don't have an Onswipe account yet, click the link to set up your account. It's very simple and takes place right inside your WordPress Admin.
   If you already have an Onswipe account, go ahead and input your username and click Save.
